# Gestion Vente

Gestion Vente est une application web de gestion commerciale conçue pour faciliter le pilotage quotidien d’une boutique ou d’un commerce. Elle permet de centraliser les ventes, les encaissements, la gestion des produits, des clients et des commandes dans une interface simple, rapide et efficace.

Cette solution est destinée aux petites et moyennes entreprises souhaitant digitaliser leur activité et améliorer le suivi de leurs opérations commerciales.

## Fonctionnalités

- Tableau de bord avec indicateurs de performance
- Gestion des ventes journalières
- Génération et impression de reçus
- Gestion des encaissements
- Gestion des produits
- Gestion des catégories
- Gestion des clients
- Gestion des commandes
- Suivi des stocks
- Historique des transactions
- Recherche et filtrage avancés
- Interface responsive (ordinateur, tablette, mobile)

## Technologies utilisées

### Backend

### Frontend
- HTML5

- JavaScript
- jQuery

## Architecture

Le projet suit une architecture modulaire afin de faciliter la maintenance, l’évolution et l’ajout de nouvelles fonctionnalités. Chaque module est organisé de manière à garantir une meilleure lisibilité du code et une séparation claire des responsabilités.

## Installation

1. Cloner le dépôt :
git clone https://github.com/kaborethierry/gestion_vente.git

2. Importer la base de données MySQL dans votre serveur local ou distant.

3. Configurer les informations de connexion à la base de données dans le fichier de configuration du projet.

4. Déployer le projet sur un serveur compatible PHP (Apache/Nginx).
